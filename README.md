# Diccionario Musical

**Diccionario Musical** es una herramienta interactiva para explorar escalas, acordes, progresiones, ritmo y secuencias MIDI desde una interfaz visual, práctica y experimental.

Nació como un diccionario de acordes, pero fue creciendo hasta transformarse en un pequeño laboratorio musical: un lugar donde la teoría no se queda escrita, sino que se puede escuchar, tocar, ordenar, repetir, modificar y exportar.

**Demo pública:**  
https://fersloc.github.io/diccionario-musical/

---

## Idea central

La idea del proyecto es simple:

> aprender música probando música.

En vez de mirar solamente definiciones, tablas o fórmulas, el Diccionario Musical permite explorar relaciones entre notas, escalas, acordes, progresiones y ritmo de una manera directa. La persona puede elegir una raíz, mirar las notas, escuchar el resultado, construir una secuencia, agregar variaciones, usar probabilidad y exportar material MIDI para seguir trabajando en un DAW como Ableton Live, FL Studio, Reaper, Logic, Bitwig u otros.

## Qué permite hacer

### Escalas

La sección de escalas permite explorar distintas escalas a partir de una tonalidad o raíz.

Incluye:

- notas de la escala;
- grados;
- receta interválica;
- teclado visual;
- acordes derivados;
- relación entre escala, color musical y armonía.

La escala funciona como la “cancha” de la música: define qué notas están disponibles y qué sensación general puede tener una idea musical.

---

### Acordes

La sección de acordes permite elegir una nota raíz y revisar distintas familias armónicas.

Incluye acordes mayores, menores, séptimas, acordes disminuidos, aumentados, suspendidos y otras variaciones.

El objetivo no es solamente mostrar nombres, sino entender qué notas forman cada acorde y cómo cambia el color emocional cuando cambia una tercera, una quinta, una séptima o una tensión.

---

### Guía acorde-escala

La guía conecta acordes con escalas compatibles.

Sirve para responder preguntas como:

- ¿qué escala puedo usar sobre este acorde?;
- ¿qué acordes salen de esta escala?;
- ¿por qué ciertas notas suenan dentro o fuera?;
- ¿cómo puedo pasar de teoría a composición?

Esta parte busca ayudar a construir puentes entre armonía, melodía e improvisación.

---

### Progresiones

La sección de progresiones permite pasar de acordes sueltos a estructuras musicales.

Incluye:

- constructor de secuencias;
- acordes y escalas agregables;
- reproducción;
- loop;
- control de BPM;
- entrada manual avanzada;
- opciones para guardar, abrir, copiar, borrar y deshacer;
- exportación MIDI.

Una progresión transforma acordes aislados en una historia armónica. No importa solamente qué acorde suena, sino hacia dónde va.

---

### Ritmo / 64 SEQ

El proyecto incorpora una zona dedicada al ritmo y a la secuenciación por pasos.

Incluye:

- secuenciador multipista;
- patrones;
- cadenas de patrones;
- acentos;
- probabilidad por step;
- asignación de notas desde escalas o acordes;
- ejemplos de patrones;
- controles de swing, humanización y velocidad aleatoria;
- exportación MIDI de batería.

Esta sección acerca el proyecto al mundo de las máquinas, cajas de ritmo, secuenciadores, Volca Drum, KeyStep, TD-3 y flujos de composición electrónica.

## Exportación MIDI

Una de las funciones centrales del proyecto es la exportación MIDI.

El archivo MIDI exportado contiene notas, duración y ritmo. No contiene el sonido del sintetizador. Eso permite llevar las ideas creadas dentro del Diccionario Musical hacia otros programas o máquinas.

Se puede usar el MIDI exportado para:

- abrirlo en Ableton Live;
- probarlo con sintetizadores;
- usarlo como base para bajos, melodías o acordes;
- modificarlo en un piano roll;
- enviarlo a hardware externo;
- construir canciones o loops más largos.

---

## Paisajes sonoros

El proyecto incluye una sección de paisajes sonoros generativos para acompañar la práctica musical.

Son ambientes creados desde el navegador, sin archivos externos, pensados para tocar encima, estudiar escalas, probar acordes o generar una atmósfera mientras se trabaja.

Ejemplos de paisajes:

- aire;
- pájaros de mañana;
- río y aves;
- bosque;
- lluvia;
- viento;
- mar.

Estos sonidos funcionan como capas lentas de fondo y quedan en loop hasta detenerlos.

## SoundLab

**SoundLab** es la próxima gran expansión conceptual del proyecto.

La idea es que el Diccionario Musical no sea solamente una herramienta para consultar escalas, acordes y ritmos, sino también un laboratorio sonoro donde distintas fuentes puedan transformarse en material musical.

SoundLab apunta a experimentar con:

- sonidos generativos desde el navegador;
- paisajes sonoros;
- análisis de archivos de audio;
- entrada desde micrófono;
- interacción táctil;
- gestos del celular;
- movimiento;
- datos convertidos en sonido;
- probabilidad, ritmo y variación;
- creación de patrones a partir de fuentes no tradicionales.

La visión de SoundLab es acercar el proyecto a una especie de instrumento digital experimental: mitad diccionario, mitad secuenciador, mitad laboratorio de sonido.

---

## Filosofía del proyecto

Este proyecto busca unir tres mundos que normalmente aparecen separados:

1. **Teoría musical**  
   Escalas, acordes, intervalos, grados, modos, progresiones y ritmo.

2. **Experimentación**  
   Escuchar, probar, equivocarse, repetir, mover notas, activar silencios, usar probabilidad y descubrir combinaciones inesperadas.

3. **Producción musical**  
   Exportar MIDI, llevar ideas a un DAW, conectar con sintetizadores, construir loops y convertir teoría en material real.

La intención no es hacer una enciclopedia fría. La intención es crear una herramienta viva para aprender haciendo.

## Cómo usarlo

1. Abrir la demo pública.
2. Elegir una sección: Escalas, Acordes, Guía, Progresiones, Ritmo o Conceptos.
3. Explorar notas, grados, acordes o patrones.
4. Reproducir el resultado.
5. Crear una secuencia o patrón.
6. Exportar como MIDI si se quiere seguir trabajando en otro programa.

---

## Notas importantes

El proyecto no pretende reemplazar un DAW ni una formación musical completa. Es una herramienta de exploración, aprendizaje y composición.

El MIDI exportado contiene información musical, pero no contiene el sonido final. El sonido dependerá del instrumento virtual, sintetizador o sampler donde se cargue después.
